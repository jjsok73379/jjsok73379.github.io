# 테이블  
{: .no_toc }

`|`와 `-` (3개 이상)의 조합으로 테이블을 만들 수 있다.  

  - 정렬  
    * 왼쪽 정렬 |:—|  
    * 오른쪽 정렬 |—:|  
    * 가운데 정렬 |:—:|  

```
|**제목**|레이팅|감상평|
```  
