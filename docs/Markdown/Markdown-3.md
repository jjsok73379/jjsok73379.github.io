# 마크다운 목록 작성법  
{: .no_toc }

## 목록  
{: .no_toc }
## - 순서있는 목록(번호)  

순서있는 목록은 숫자와 점을 사용한다.  
```
1. 첫번째
2. 두번째
3. 세번째
```  
1. 첫번째  
2. 두번째  
3. 세번째  

**현재까지는 어떤 번호를 입력해도 순서는 내림차순으로 정의된다.**  
```
1. 첫번째
3. 세번째
2. 두번째
```  
1. 첫번째  
3. 세번째  
2. 두번째  

## - 순서없는 목록(글머리 기호: `*`, `+`, `-` 지원)  
```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```  
* 빨강
  * 녹색
    * 파랑  

+ 빨강
  + 녹색
    + 파랑  

- 빨강
  - 녹색
    - 파랑  

혼합해서 사용하는 것도 가능하다  
```
* 1단계
  - 2단계
    + 3단계
      + 4단계
```  
* 1단계
  - 2단계
    + 3단계
      + 4단계
