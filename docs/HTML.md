## HTML이란?  
{: .no_toc }
**HyperText** : 하이퍼링크를 통해 어떤 문서에서 다른 문서로 접근할 수 있는 텍스트  
**Markup** : (콘텐츠를) 표시하다  
**Language** : 언어  
=> 하이퍼 텍스트와 콘텐츠를 표시해주는 언어!
